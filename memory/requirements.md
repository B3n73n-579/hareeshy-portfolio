# Requirements

## Functional
1. Landing page with hero, 3D particle background, scroll storytelling
2. About page with bio, skills radar visualization
3. Projects page with sticky scrolling experience cards
4. Experience timeline (interactive, scroll-driven)
5. Technical blog (MDX, categories, tags, search, pagination, RSS)
6. Architecture gallery with 3D interactive diagrams
7. AI Lab showcase with interactive demos
8. Contact form with backend API + email notification
9. Resume download (PDF)
10. Admin CMS for blog posts

## Non-Functional

- LCP <2s, CLS <0.1, INP <200ms
- 60fps 3D on desktop, 30fps on mobile
- 90%+ test coverage
- WCAG 2.1 AA compliance
- Full responsive: mobile, tablet, desktop
