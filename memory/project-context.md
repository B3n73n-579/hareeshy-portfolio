# Project Context

## Overview
Personal brand platform and technical blog for Hareesh Y, a Senior Java Developer specializing in microservices, cloud-native architecture, and AI-powered applications.

## Domain
- Engineering portfolio & personal branding
- Technical blog (MDX-based)
- AI Lab for experimental AI/LLM applications
- Contact & CMS system

## Tech Stack
- **Frontend:** React 19, TypeScript, Vite, TailwindCSS, Framer Motion, R3F
- **Backend:** Java 21, Spring Boot 3, PostgreSQL, Redis, Kafka
- **Deployment:** Vercel (frontend), Railway (backend), Docker Compose (local)

## Design System
- Theme: Luxury Cyberpunk Engineering
- Colors: #050816 bg, #00E5FF primary, #7C4DFF secondary, #FF00E5 accent
- Typography: Satoshi/Geist headings, Inter body, JetBrains Mono code
